About this Dashboard

This dashboard displays real-time sensor readings from an IoT device, including temperature and humidity data. The data is automatically fetched from a Google Sheets CSV and updated every 30 seconds.

Features: Live data visualization, last 30 readings display, interactive charts with hover details, and automatic refresh functionality.
